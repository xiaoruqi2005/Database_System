# Lab 3 实验完成流程

## 1. 实验目标
根据 lab3.md，实现有符号 BIGINT 字段（8字节，long long），支持增删改查，超出范围输出 failure。

## 2. 需求拆解
- 新增 TYPE_BIGINT 类型
- 修改 lexer/parser 支持 BIGINT 关键字和大整数字面量
- 修改 Value 支持 8 字节存储和序列化
- 修改 analyzer 支持溢出检测和隐式类型转换
- 修改所有 switch(type) 分支

## 3. 实现思路
### 3.1 类型系统
- ColType 新增 TYPE_BIGINT，SvType 新增 SV_TYPE_BIGINT

### 3.2 整数字面量解析与溢出检测
- lexer 将整数字面量以字符串形式存储
- IntLit 构造函数接收字符串，用 strtoll 解析，保留 raw_str
- analyzer 通过字符串比较检测 BIGINT 范围溢出

### 3.3 Value 扩展
- union 新增 bigint_val，init_raw 新增 TYPE_BIGINT 分支

### 3.4 类型兼容性
- INT/BIGINT/FLOAT 之间允许隐式转换
- INSERT/UPDATE/WHERE 路径均添加转换逻辑

## 4. 修改文件明细
- src/defs.h: ColType 新增 TYPE_BIGINT
- src/parser/ast.h: SvType 新增 SV_TYPE_BIGINT; IntLit 改为 long long + raw_str
- src/parser/lex.l: 新增 BIGINT 关键字; 整数字面量存为字符串
- src/parser/yacc.y: VALUE_INT 改为 sv_str; 新增 BIGINT 类型规则
- src/common/common.h: Value 新增 bigint_val 和 set_bigint
- src/analyze/analyze.cpp: convert_sv_value 添加溢出检测; 类型兼容性扩展
- src/optimizer/planner.h: interp_sv_type 新增 BIGINT 映射
- src/index/ix_index_handle.h: ix_compare 新增 TYPE_BIGINT
- src/execution/executor_seq_scan.h: compare_value 新增 TYPE_BIGINT
- src/execution/executor_nestedloop_join.h: compare_value 新增 TYPE_BIGINT
- src/execution/executor_insert.h: INSERT 路径添加隐式转换
- src/execution/execution_manager.cpp: select_from 新增 TYPE_BIGINT 输出
